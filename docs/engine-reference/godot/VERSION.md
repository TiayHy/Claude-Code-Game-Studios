## 知识差距警告

LLM 的训练数据很可能只覆盖到 Godot 4.3 左右。4.4、4.5 和 4.6 版本引入了重大变化，模型对此**并不知晓**。
在建议 Godot API 调用之前，务必查阅本目录进行交叉验证。

## 截止日期后的版本时间线

| 版本 | 发布时间 | 风险等级 | 主题 |
|---------|---------|------------|-----------|
| 4.4 | 约 2025 年中 | 中等 | Jolt 物理选项、FileAccess 返回类型、着色器纹理类型变更 |
| 4.5 | 约 2025 年末 | 高 | 无障碍（AccessKit）、可变参数、@abstract、着色器烘焙、SMAA |
| 4.6 | 2026 年 1 月 | 高 | Jolt 默认为真、发光重构、Windows D3D12 默认为真、IK 恢复 |
